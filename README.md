# Programmers 코딩테스트 풀이

한국폴리텍대학 재학 중 [프로그래머스](https://programmers.co.kr) 코딩테스트 문제를 풀며
알고리즘과 기초 문법을 연습한 저장소입니다.

## 구성

| 언어 | 경로 | 비고 |
|------|------|------|
| Java | `java/` | 메인 풀이 언어 |
| C | `c/` | 동일 문제 다국어 풀이 |
| Python | `python/` | 동일 문제 다국어 풀이 |

## 다룬 내용 (Level 0)

배열 평균·중앙값·최빈값, 사칙연산, 몫/나머지 계산, 분수 덧셈,
홀수 배열 생성, 피자 조각 계산 등 기초 수학·배열 문제

## 학습 포인트

> 하나의 문제를 **Java·C·Python으로 각각** 풀어보며 언어별 문법과 접근 방식의
> 차이를 체득하는 데 중점을 뒀습니다.
